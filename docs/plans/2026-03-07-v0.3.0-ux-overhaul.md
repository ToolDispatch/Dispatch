## Task 1: stdout injection in dispatch.sh

**Files:** `dispatch.sh`

**What:** Replace all `/dev/tty` output with stdout. CC UserPromptSubmit hook stdout is injected into Claude's context automatically.

**Step 1: Remove /dev/tty rendering block**

Delete the entire Python `PYEOF` render block (lines ~339-408) and replace with a stdout print:

```bash
# ── Output notification to stdout (injected into Claude context by CC) ──────
python3 - "$TASK_TYPE" "$RECOMMENDATIONS" "$CONFIDENCE" <<'PYEOF'
import json, sys

task_type = sys.argv[1]
try:
    recs = json.loads(sys.argv[2])
except Exception:
    recs = {"installed": [], "suggested": []}

confidence = float(sys.argv[3]) if len(sys.argv) > 3 and sys.argv[3] else 0.0
conf_label = "high" if confidence >= 0.85 else "medium"
task_display = task_type.replace('-', ' ').title()

installed = recs.get("installed", [])
suggested = recs.get("suggested", [])

lines = [
    f"[DISPATCH] Task shift detected: {task_display} ({conf_label} confidence)",
    "",
]

if installed:
    lines.append("Recommended tools (already installed):")
    for p in installed:
        lines.append(f"  + {p['name']}")
        if p.get('reason'):
            lines.append(f"    Why: {p['reason']}")

if suggested:
    lines.append("")
    lines.append("Suggested tools (not installed):")
    for s in suggested:
        lines.append(f"  ↓ {s['name']}")
        if s.get('reason'):
            lines.append(f"    Why: {s['reason']}")
        if s.get('install_cmd'):
            lines.append(f"    Install: {s['install_cmd']}")

lines.append("")
lines.append("Please acknowledge these recommendations before answering my question. Ask which (if any) I want to activate or install, then proceed.")

print('\n'.join(lines))
PYEOF
```


**Step 2: Add log_recommendations() function to db.py**

```python
def log_recommendations(token, task_type, installed, suggested):
    try:
        with get_conn() as conn:
            with conn.cursor() as cur:
                cur.execute(
                    "INSERT INTO recommendations_log (token, task_type, installed_shown, suggested_shown) VALUES (%s, %s, %s, %s)",
                    (str(token), task_type, json.dumps(installed), json.dumps(suggested))
                )
            conn.commit()
    except Exception:
        pass
```
